五子棋
===

### dom 版本

``` bash

cd ./dom

# install dependencies
npm install

# serve with hot reload at localhost:8003
npm run dev

# build for production with minification
npm run build

# lint
npm run lint

```

### canvas 版本

``` bash

cd ./canvas

# install dependencies
npm install

# serve with hot reload at localhost:8004
npm run dev

# build for production with minification
npm run build

# lint
npm run lint

```
